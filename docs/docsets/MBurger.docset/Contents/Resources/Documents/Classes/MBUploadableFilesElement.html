MBUploadableFilesElement

@interface MBUploadableFilesElement : MBUploadableElement

An uploadable element representing some files. When converted to form with -[MBUploadableElement toForm] is returned an array with objects with name elements[locale][name][index] and value the file in which the image is saved.
