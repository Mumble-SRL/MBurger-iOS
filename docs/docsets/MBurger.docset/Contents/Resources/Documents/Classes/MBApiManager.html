MBApiManager

@interface MBApiManager : NSObject

This class is responsible to make all the networking requests to the MBurger Api, MBManager, MBClient and MBAdmin are built above this class.

  • +callApiWithApiToken:Locale:ApiName:HTTPMethod:Parameters:HeaderParameters:Development:Success:Failure:

    Calls the api to the MBurger endpoint with the parameters specified. At the end of the network call it will be called the success block or the failure block, if passed.

    Parameters

    apiToken

    The api token used to call the MBurger Api.
    locale

    The locale passed to the api call.
    apiName

    The name of the api to call.
    httpMethod

    The http method used to call the api.
    parameters

    The parameters passed with the api call.
    headerParameters

    The parameters passed in the header of the request with the api call.
    development

    If it’s in development mode.
    success

    A block object to be executed when the task finishes successfully. This block has no return value and takes one argument: the response of the api (a MBResponse).
    failure

    A block object to be executed when the task finishes unsuccessfully, or that finishes successfully, but the server encountered an error. This block has no return value and takes one argument: the error describing the error that occurred.
